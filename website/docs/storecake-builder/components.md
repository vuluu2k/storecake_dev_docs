# Components

## My storecake components base in ant-design-vue version 3

Inherit props from antdv + new props — see [Storecake components docs](https://vuluu2k.github.io/storecake_components).

- Components import từ `@/components/design`
- Example:

```js
import Button from '@/components/design/Button.vue'
```

> Sau này sẽ convert thành package npm.
